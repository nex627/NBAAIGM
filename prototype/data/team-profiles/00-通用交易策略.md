# NBA 交易底层通用策略

> 适用所有球队 · 最后更新：2026-05-24

## 一、核心原则（所有交易决策的底层逻辑）

### 1. 价值匹配原则
任何交易的核心都是**价值对等**。总价值 = 当前战力价值 + 未来潜力价值 + 合同价值。
- 没有绝对的"非卖品"，只有**筹码不足的交易**
- 建队核心球员的交易门槛极高（通常需要全明星级回报 + 高顺位选秀权 + 年轻潜力）
- 重建球队的高潜力新秀交易门槛同样极高

### 2. 建队阶段导向原则
不同阶段的球队，交易逻辑完全不同：

| 阶段 | 核心目标 | 交易倾向 |
|------|---------|---------|
| **争冠球队** | 赢在当下 | 愿意送出未来资产换取即战力，优先考虑季后赛经验、防守、关键球能力 |
| **季后赛球队** | 保持竞争力 + 适度补强 | 平衡当下与未来，寻求性价比交易 |
| **重建球队** | 积累资产 + 培养年轻球员 | 愿意吃进垃圾合同换取选秀权，送走即战力换取未来资产 |

### 3. 风险分层原则
球员按交易风险和价值分层：

| 层级 | 定义 | 交易难度 | 典型筹码要求 |
|------|------|---------|-------------|
| **建队核心** | 联盟前20级别的球星，球队战术和文化核心 | 极高 | 全明星级球员 + 高顺位首轮 + 年轻潜力 |
| **优质首发** | 联盟前60级别的球员，稳定贡献 | 中高 | 同级别球员 + 或多个优质筹码组合 |
| **轮换球员** | 稳定的替补或边缘首发 | 中等 | 同级别球员或一个优质筹码 |
| **到期合同** | 合同即将到期的功能性球员 | 低 | 容易出手，常作为交易配平 |
| **溢价合同** | 表现与薪资不匹配的长期合同 | 高 | 通常需要搭上选秀权才能送走 |

### 4. 合同为王原则
- **到期合同**是优质筹码（释放空间、交易配平、不影响长期灵活性）
- **新秀合同**是黄金资产（低薪高能、可控性强）
- **长期溢价合同**是负资产（限制操作空间、降低交易灵活性）
- **鸟权**可以超越工资帽续约，是重要的交易杠杆

### 5. 适配性原则
球员之间的**打法适配、教练体系适配、球队文化适配**，比单纯的天赋堆积更重要。
- 双塔组合在季后赛可能被空间型球队针对
- 持球大核心需要空间型队友拉开空间
- 防守型球队需要纪律性强的球员

### 6. 时机原则
三大关键交易窗口：
1. **交易截止日**（赛季中期）：最后的补强机会，卖家多、议价空间大
2. **选秀夜**：选秀权价值最高的时刻，大量交易发生
3. **自由市场前**：清理空间、签换交易的高峰期

## 二、通用决策框架（任何交易都可以套用）

### 交易前必问5个问题
1. **为什么要做这笔交易？**（短期目标 / 长期目标 / 财务目标）
2. **我们放弃的是什么？**（当前战力 / 未来潜力 / 灵活性 / 化学反应）
3. **我们得到的是什么？**（当前战力 / 未来潜力 / 灵活性 / 合同价值）
4. **风险是什么？**（伤病 / 合同 / 适配性 / 更衣室影响）
5. **有没有更好的替代方案？**（自由市场 / 内部挖掘 / 其他交易方案）

### 否决交易的红线
- 送出的价值明显高于得到的价值，且没有战略层面的补偿
- 交易后长期灵活性被彻底锁死（如吞下多个长期大合同）
- 核心球员不开心且无法安抚（更衣室毒药）
- 违反薪资配平规则或硬帽限制

## 三、AI 经理决策指南

作为 GM，请遵循以下决策逻辑：

1. **先判断球队阶段**：争冠 / 季后赛 / 重建？
2. **再评估球员层级**：建队核心 / 优质首发 / 轮换 / 到期 / 溢价？
3. **然后考虑适配性**：球员打法是否匹配教练体系？更衣室是否兼容？
4. **最后评估价值**：筹码是否足够？有没有更好的方案？

### 关于"非卖品"的正确表述
- ❌ 错误："绝对不交易 XX"
- ✅ 正确："XX 是球队核心，交易门槛极高，需要全明星级回报 + 高顺位选秀权才会考虑"
- ✅ 正确："作为重建球队，我们的高潜力新秀除非收到无法拒绝的报价，否则不会交易"

### 关于"愿意送出"的正确表述
- ❌ 错误："愿意送出 XX"（暗示主动兜售）
- ✅ 正确："XX 是可交易资产，如果报价合理会考虑"
- ✅ 正确："如果能换回更适合球队的资产，我们愿意讨论任何球员的交易"
